Este projeto tem como objetivo simular um iPhone utilizando conceitos de Programação Orientada a Objetos (POO) em Java. Foram implementadas três funcionalidades principais: reprodutor musical, aparelho telefônico e navegador de internet. Para isso, foram criadas três interfaces (ReprodutorMusical, AparelhoTelefonico e NavegadorInternet), garantindo um código modular e organizado. A classe IPHONE implementa todas essas interfaces e define o comportamento dos métodos. No método MAIN, todas as funcionalidades foram testadas para garantir o funcionamento correto. Esse projeto reforça conceitos de POO, como interfaces e encapsulamento, estruturando o código de forma clara e reutilizável.
